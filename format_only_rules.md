# 格式规范（精简版）

## 1. 首段格式

- **单义词格式**：首段 = 主语 + is/are + `***核心定义***` + 句号
- **首段禁止第二句（铁律）**：首段只能有一句话，`***` 闭合后的句号就是首段结尾，句号后不能再有任何内容
- **高亮完整性**：`***` 必须包住首段句号前的全部定义内容
- **冠词强制包含**：冠词 (`a`, `an`, `the`) 必须在 `***` 内部
- **系动词隔离**：系动词 (`is`, `are`, `was`, `refer to`) 必须在 `***` 外部
- **首段高亮结尾标点**：如果高亮以引号结尾，句号在引号内，然后闭合 `***`（如 `***..."Title."***`）
- **首段主语引号规则**：只有作品名才加引号，人名、地名、品牌名不加引号

## 2. 列表格式

- **四级标题后强制列表**：`#### Title` 下方必须直接跟列表（`-` 或 `1.`），严禁段落
- **小标题必须加粗**：列表项必须以 `**Title**: ...` 开头
- **小标题大小写**：遵循 Title Case，每个单词首字母大写（介词、冠词除外）
- **列表内容完整句**：小标题后的内容必须是完整句子，不能省略主语
- **句号规则**：列表项末尾必须加句号
- **单项内容处理**：四级标题下只有1项时，写成段落形式，不用列表符号
- **二级列表规则**：如果一级列表项下有二级列表，一级标题后只能有冒号，不能有其他内容
- **四级标题内聚性**：四级标题下的列表项必须属于同一类别，禁止将差异过大的内容强行归入同一标题

## 3. 标点符号

- **引号内标点**：逗号和句号必须在引号内
- **禁止分号连接句子**：禁止用 `;` 连接多个句子，应改用句号

## 4. 星号/反引号

- **正文中星号/反引号改双引号**：`*text*`、`**text**`、`` `text` `` 必须改为 `"text"`
- 列表小标题的加粗（`- **Title**:`）不受影响

## 5. 引用位置

- **强制移至段末**：Note 引用必须移到段落最后，格式为 `内容.[Note X](#)`
- **四级标题下列表必须有引用**：每个列表项都必须有至少一个 Note 引用

## 6. 正文内容禁止加粗

- 除列表小标题外，正文内容不能用双星号加粗（人名、地名、作品名等都不加粗）
